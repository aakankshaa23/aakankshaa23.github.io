Share Your File

An Android Application used to share file between 2 devices without internet. Using sockets and I/O Stream in Android, 2 devices connected on same IP can share files between them. It includes client-server based architecture, where sender acts as server. It also includes a QR Code which the sender generates and can be scanned by receiver, to reduce manual task of entering IP.

Entry Made Easy

An Application to effectively manage entries made at any place. We need to submit our entries at various entry points in our college. Hence, in an attempt to simplify this task, I made this app in which manually handled entries are automated. It fetches roll number from library card picture, and then automatically does entry in database. When student returns back, again scanning roll number will close the entry. Made using Firebase, Vision Api, it includes text recognition to recognise text from ID Card

Sports Management System

The main aim of this project to avoid the tedious manual work for managing the sports event in any Institute. An Android app to register and solve complaints of employees along with the feature to see the status of complaint. Implemented using Firebase and Android Studio. User can register themselves using Google/Normal registration. On logging inside app, user can register complaint relating to a dept. There is admin panel which can see all the complaints and can forward a complaint to other dept as email or mark as resolved.

Lets Chat

A basic chatting Mobile Application made using Firebase including many features. User can see all registered users in app alongwith status whether they are online or not. On clicking any user, a chat window will open, where user can chat. User can also see whether the message is delivered or seen. Implemented using Firebase, it contain Firebase Realtime Database to store and retrieve messages, various android components and material design for frontend .
